# Z Monitor Switcher v0.1.9

## English

### Highlights

- Added Linux Wayland shortcut support through the desktop shortcut
  authorization flow. X11, macOS, and Windows continue to use the existing
  native shortcut path.
- Improved input-source switching reliability by serializing repeated switch
  requests and keeping the latest requested source.
- Improved Ubuntu desktop behavior: closing the main window now keeps the app
  available from the tray, and tray controls open near the tray/menu location.
- Improved Linux updates for DEB installations so the updater can select the
  signed DEB package route.

### Linux / Ubuntu

- Wayland shortcut setup now uses the system dialog and displays the shortcut
  returned by the desktop.
- Shortcut activations still use the same KVM confirmation flow before sleep or
  shutdown actions run.
- Ubuntu tray menus provide access to quick controls, the main window, and quit
  even when the desktop shell does not send tray click events.

### Notes

- Linux support targets Ubuntu 22.04 X11. Monitor compatibility still depends on
  DDC/CI support, `/dev/i2c-*` permissions, and the desktop environment.
- Wayland shortcuts require a desktop environment that supports the Global
  Shortcuts Portal. X11 remains available for desktops that do not provide it.
- macOS builds target Apple Silicon and require macOS 12.0 or later.
- macOS builds are not signed or notarized yet; use right-click Open or the
  documented quarantine fallback if needed.
- Windows builds are unsigned and may show Microsoft SmartScreen.

## 简体中文

### 亮点

- 新增 Linux Wayland 快捷键支持，通过桌面系统的快捷键授权流程完成配置。X11、macOS
  和 Windows 继续使用原有的本机快捷键路径。
- 提升输入源切换可靠性：连续点击时会串行处理切换请求，并保留最后一次想切换的输入源。
- 改进 Ubuntu 桌面体验：关闭主窗口后应用仍保留在托盘中，托盘快捷控制会在托盘/菜单位置附近打开。
- 改进 Linux 自动更新路径，DEB 安装用户可匹配到签名的 DEB 更新包。

### Linux / Ubuntu

- Wayland 快捷键配置现在使用系统弹窗，并显示桌面系统返回的最终快捷键。
- 快捷键触发输入源切换时，仍会先进入原有 KVM 确认流程，再执行休眠或关机等后置动作。
- Ubuntu 托盘菜单提供快捷控制、显示主窗口和退出入口，即使桌面环境不发送托盘点击事件也能使用。

### 注意事项

- Linux 支持目标是 Ubuntu 22.04 X11。显示器兼容性仍取决于 DDC/CI 支持、
  `/dev/i2c-*` 权限以及桌面环境。
- Wayland 快捷键需要桌面环境支持 Global Shortcuts Portal；不支持的桌面仍可使用 X11 会话。
- macOS 构建面向 Apple Silicon，要求 macOS 12.0 或更高版本。
- macOS 构建暂未签名/公证；如有需要，请使用右键打开或文档中的 quarantine
  备用命令。
- Windows 构建暂未签名，可能出现 Microsoft SmartScreen 提示。
