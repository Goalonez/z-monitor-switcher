# Z Monitor Switcher v0.1.13

## English

### Highlights

- Retargeted the active Linux support baseline and release build runner to
  Ubuntu 24.04 X11.
- Reviewed Ubuntu 24.04 Linux dependencies and kept the existing WebKitGTK 4.1,
  GTK 3, Ayatana AppIndicator, and libudev package families.
- Improved Ubuntu 24.04 X11 shortcut recording by temporarily clearing existing
  app hotkeys while recording, then restoring or re-applying configured hotkeys
  after cancellation or save.
- Made unsupported Linux shortcut sessions fail visibly instead of presenting an
  in-app recorder that cannot work.
- Cleaned up failed Wayland Global Shortcuts Portal attempts by closing only the
  newly created session while preserving the previous valid shortcut session.

### Notes

- Linux support targets Ubuntu 24.04 X11. Monitor compatibility still depends on
  DDC/CI support, `/dev/i2c-*` permissions, and the desktop environment.
- Ubuntu 24.04 X11 runtime verification for native shortcut recording,
  restart-time re-apply, and shortcut-triggered input switching should be
  performed with this release's GitHub Actions artifacts.
- Wayland shortcuts remain best-effort through the desktop Global Shortcuts
  Portal. Desktops without Portal support should show a clear error; X11 remains
  the required supported shortcut baseline.
- macOS builds target Apple Silicon and require macOS 12.0 or later.
- macOS builds are not signed or notarized yet; use right-click Open or the
  documented quarantine fallback if needed.
- Windows builds are unsigned and may show Microsoft SmartScreen.

## 简体中文

### 亮点

- 将当前 Linux 支持基线和发布构建 runner 调整为 Ubuntu 24.04 X11。
- 复核 Ubuntu 24.04 Linux 依赖后，保留现有 WebKitGTK 4.1、GTK 3、Ayatana
  AppIndicator 和 libudev 依赖包族。
- 改进 Ubuntu 24.04 X11 快捷键录制：录制期间临时清除应用已有全局快捷键，
  取消或保存后再恢复/重新应用已配置快捷键。
- 不支持全局快捷键的 Linux 会话会显示明确错误，不再呈现一个实际不可用的应用内录制入口。
- Wayland Global Shortcuts Portal 配置失败时只清理新建 session，保留之前有效的快捷键 session。

### 注意事项

- Linux 支持目标是 Ubuntu 24.04 X11。显示器兼容性仍取决于 DDC/CI 支持、
  `/dev/i2c-*` 权限以及桌面环境。
- 本版本的 Ubuntu 24.04 X11 实机验证（本机快捷键录制、重启后重新应用、快捷键触发输入源切换）
  建议使用 GitHub Actions 构建产物进行测试。
- Wayland 快捷键继续通过桌面 Global Shortcuts Portal 做 best-effort 支持；
  不支持 Portal 的桌面应显示明确错误，X11 仍是本次必须支持的快捷键基线。
- macOS 构建面向 Apple Silicon，要求 macOS 12.0 或更高版本。
- macOS 构建暂未签名/公证；如有需要，请使用右键打开或文档中的 quarantine
  备用命令。
- Windows 构建暂未签名，可能出现 Microsoft SmartScreen 提示。
