# Z Monitor Switcher v0.1.3

## English

### Fixes

- Fixed Windows tray icon rendering by using the packaged app icon instead of
  the macOS menu-bar template asset.
- Fixed tray popup placement on Windows so the compact panel opens next to the
  tray icon and avoids covering the taskbar.
- Improved Windows input-source switching reliability with a stronger VCP 0x60
  retry path.
- Allowed existing shortcut bindings to be re-recorded directly without clearing
  them first.
- Hid the PowerShell console window used by Windows native-control probing.

### Packaging

- Explicitly preserved project icons in platform-specific Tauri bundle configs.
- Set the Windows NSIS installer and uninstaller icons to the project icon.

### Notes

- macOS builds target Apple Silicon and require macOS 12.0 or later.
- macOS builds are not signed or notarized yet; use right-click Open or the
  documented quarantine fallback if needed.
- Windows builds are unsigned and may show Microsoft SmartScreen.

## 简体中文

### 修复

- 修复 Windows 托盘图标渲染问题，改用打包应用图标，不再使用 macOS 菜单栏模板图。
- 修复 Windows 托盘弹窗位置，使紧凑控制面板贴近托盘图标并避开任务栏。
- 加强 Windows 输入源切换可靠性，为 VCP 0x60 使用更稳的重试路径。
- 优化快捷键录制，已有快捷键可直接重新录入，无需先清除。
- 隐藏 Windows 本机控制探测时的 PowerShell 控制台窗口。

### 打包

- 在平台级 Tauri 打包配置中显式保留项目图标。
- 将 Windows NSIS 安装器和卸载器图标设置为项目图标。

### 注意事项

- macOS 构建面向 Apple Silicon，要求 macOS 12.0 或更高版本。
- macOS 构建暂未签名/公证；如有需要，请使用右键打开或文档中的 quarantine
  备用命令。
- Windows 构建暂未签名，可能出现 Microsoft SmartScreen 提示。
