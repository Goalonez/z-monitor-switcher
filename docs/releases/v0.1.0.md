# Z Monitor Switcher v0.1.0

## English

Initial public release.

### Highlights

- Switch external monitor input sources through DDC/CI.
- Adjust external monitor brightness and monitor speaker volume where supported.
- Control local system volume, plus built-in screen brightness on Windows laptops.
- Use the menu bar or system tray quick panel for compact switching and adjustments.
- Configure per-monitor input source values and optional global shortcuts.
- Enable the guarded "shutdown after switch" KVM handoff flow with a cancelable confirmation dialog.

### Notes

- macOS builds target Apple Silicon and require macOS 12.0 or later.
- macOS builds are not signed or notarized yet; use right-click Open or the documented quarantine fallback if needed.
- Windows builds are unsigned and may show Microsoft SmartScreen.

## 简体中文

首次公开发布。

### 主要内容

- 通过 DDC/CI 切换外接显示器输入源。
- 在显示器支持时调节外接显示器亮度和内置扬声器音量。
- 支持本机系统音量控制，并在 Windows 笔记本上支持内置屏亮度控制。
- 提供菜单栏/系统托盘快捷面板，用于紧凑切换和调节。
- 支持按显示器配置输入源控制值，并可为常用输入源设置全局快捷键。
- 支持带可取消确认倒计时的“切换后关机”KVM 交接流程。

### 注意事项

- macOS 构建面向 Apple Silicon，要求 macOS 12.0 或更高版本。
- macOS 构建暂未签名/公证；如有需要，请使用右键打开或文档中的 quarantine 备用命令。
- Windows 构建暂未签名，可能出现 Microsoft SmartScreen 提示。
