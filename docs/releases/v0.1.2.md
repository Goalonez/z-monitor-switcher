# Z Monitor Switcher v0.1.2

## English

### New Features

- **Menu-bar popup controls**: Open a compact control panel from the tray/menu
  bar for quick brightness, volume, and input switching without opening the main
  window.

### Maintenance

- Updated the release workflow to use Node.js 24-compatible GitHub Actions for
  pnpm setup and artifact upload/download.
- Pinned the Windows release build to an explicit runner image for more
  predictable NSIS packaging.

### Notes

- macOS builds target Apple Silicon and require macOS 12.0 or later.
- macOS builds are not signed or notarized yet; use right-click Open or the
  documented quarantine fallback if needed.
- Windows builds are unsigned and may show Microsoft SmartScreen.

## 简体中文

### 新功能

- **菜单栏弹窗控制**：从托盘/菜单栏打开紧凑控制面板，无需打开主窗口即可快速调节亮度、音量并切换输入源。

### 维护

- 更新发布工作流中的 GitHub Actions，使 pnpm setup 和产物上传/下载步骤使用
  Node.js 24 兼容版本。
- 将 Windows 发布构建固定到明确的 runner 镜像，降低 NSIS 打包环境漂移风险。

### 注意事项

- macOS 构建面向 Apple Silicon，要求 macOS 12.0 或更高版本。
- macOS 构建暂未签名/公证；如有需要，请使用右键打开或文档中的 quarantine
  备用命令。
- Windows 构建暂未签名，可能出现 Microsoft SmartScreen 提示。
