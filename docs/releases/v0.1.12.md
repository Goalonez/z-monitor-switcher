# Z Monitor Switcher v0.1.12

## English

### Highlights

- Retargeted Linux release builds and documentation to Ubuntu 22.04 X11,
  including the GitHub Actions runner and `libfuse2` dependency used for Linux
  packaging.
- Simplified Linux shortcut recording so Ubuntu 22.04 X11 no longer suspends
  and re-registers native global shortcuts for every recording session.
- Restored vertical brightness and volume sliders to a rotated horizontal range
  implementation for better WebKitGTK/X11 click and drag behavior.
- Kept Linux close-to-tray window restore simpler and more stable by avoiding
  restore-time repositioning of the hidden main window.
- Centered the main window on open and removed the outdated Ubuntu smoke-test
  checklist in favor of release-artifact validation notes.

### Notes

- Linux support targets Ubuntu 22.04 X11. Monitor compatibility still depends on
  DDC/CI support, `/dev/i2c-*` permissions, and the desktop environment.
- Ubuntu 22.04 X11 runtime verification for slider visuals, titlebar controls,
  native shortcuts, tray restore, and launch-at-login should be performed with
  this release's GitHub Actions artifacts.
- Wayland shortcuts require a desktop environment that supports the Global
  Shortcuts Portal. X11 remains available for desktops that do not provide it.
- macOS builds target Apple Silicon and require macOS 12.0 or later.
- macOS builds are not signed or notarized yet; use right-click Open or the
  documented quarantine fallback if needed.
- Windows builds are unsigned and may show Microsoft SmartScreen.

## 简体中文

### 亮点

- 将 Linux 发布构建和文档目标调整为 Ubuntu 22.04 X11，包括 GitHub Actions
  runner 以及 Linux 打包使用的 `libfuse2` 依赖。
- 简化 Linux 快捷键录制流程：Ubuntu 22.04 X11 下录制快捷键时不再每次暂停并重新注册本机全局快捷键。
- 亮度和音量的竖向滑块改回旋转横向 range 实现，改善 WebKitGTK/X11 下点击和拖动的稳定性。
- 精简 Linux 关闭到托盘后的窗口恢复路径，避免恢复隐藏主窗口时重新定位窗口。
- 主窗口打开时居中，并移除过期的 Ubuntu smoke-test 清单，改以发布产物验证说明为准。

### 注意事项

- Linux 支持目标是 Ubuntu 22.04 X11。显示器兼容性仍取决于 DDC/CI 支持、
  `/dev/i2c-*` 权限以及桌面环境。
- 本版本的 Ubuntu 22.04 X11 实机验证（滑块视觉、标题栏按钮、本机快捷键、托盘恢复、开机自启）
  建议使用 GitHub Actions 构建产物进行测试。
- Wayland 快捷键需要桌面环境支持 Global Shortcuts Portal；不支持的桌面仍可使用 X11 会话。
- macOS 构建面向 Apple Silicon，要求 macOS 12.0 或更高版本。
- macOS 构建暂未签名/公证；如有需要，请使用右键打开或文档中的 quarantine
  备用命令。
- Windows 构建暂未签名，可能出现 Microsoft SmartScreen 提示。
