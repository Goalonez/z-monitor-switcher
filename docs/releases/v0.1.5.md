# Z Monitor Switcher v0.1.5

## English

### Improvements

- Added a per-switch shutdown choice when "shutdown after switch" is enabled.
  Input-source buttons and global hotkeys now open the confirmation flow before
  switching away on the configured trigger value.
- Choosing "Shut down" switches the monitor input and then shuts down this
  machine; choosing "Do not shut down" switches the monitor input without
  shutting down.
- Updated input-source buttons to keep a lighter selected state with press
  feedback instead of turning into a dark primary button.
- Changed launch-at-login to start silently without showing the main window,
  and migrates existing enabled autostart registrations to the silent-start
  configuration.
- Improved Windows tray popup first-show placement by waiting for the compact
  panel's initial size before positioning and showing it.

### Packaging

- Refreshed packaged app and installer icon assets to improve Windows icon
  clarity and sizing.

### Notes

- macOS builds target Apple Silicon and require macOS 12.0 or later.
- macOS builds are not signed or notarized yet; use right-click Open or the
  documented quarantine fallback if needed.
- Windows builds are unsigned and may show Microsoft SmartScreen.

## 简体中文

### 改进

- 开启“切换后关机”后，输入源按钮和全局快捷键在命中配置的触发值时，会先打开确认流程，再执行切换。
- 选择“关机”会先切换显示器输入源，再关闭本机；选择“不关机”则只切换输入源，不执行关机。
- 优化输入源按钮选中态：保留更轻量的选中样式和点击反馈，不再变成深色主按钮。
- 开机启动改为静默启动，不再显示主窗口，并会把已开启的旧 autostart 配置迁移到静默启动配置。
- 改进 Windows 托盘弹窗首次展示位置：等待紧凑面板完成初始尺寸上报后再定位并显示，减少遮挡托盘/任务栏的问题。

### 打包

- 刷新应用和安装器图标资源，改善 Windows 图标清晰度和尺寸表现。

### 注意事项

- macOS 构建面向 Apple Silicon，要求 macOS 12.0 或更高版本。
- macOS 构建暂未签名/公证；如有需要，请使用右键打开或文档中的 quarantine
  备用命令。
- Windows 构建暂未签名，可能出现 Microsoft SmartScreen 提示。
