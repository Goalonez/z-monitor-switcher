# Z Monitor Switcher v0.1.4

## English

### Fixes

- Fixed Windows tray popup positioning after the compact panel auto-resizes, so
  it stays outside the taskbar/tray area.
- Rebuilt the app, installer, tray, and DMG icon assets from a transparent logo
  source to avoid black or opaque edges on Windows.
- Improved Windows input-source switching by replacing the high-level DDC
  wrapper with direct dxva2 physical-monitor calls and stronger VCP 0x60 retry
  behavior.
- Kept input switching behavior per-monitor in the current UI design.

### Packaging

- Restored the transparent logo to the normal `src/assets/logo.png` asset path.
- Verified the macOS app bundle points to `icon.icns`, and the DMG build inputs
  include an alpha icon for the mounted volume icon.

### Notes

- macOS builds target Apple Silicon and require macOS 12.0 or later.
- macOS builds are not signed or notarized yet; use right-click Open or the
  documented quarantine fallback if needed.
- Windows builds are unsigned and may show Microsoft SmartScreen.

## 简体中文

### 修复

- 修复紧凑面板自动调整尺寸后的 Windows 托盘弹窗定位，使其避开任务栏/托盘区域。
- 使用透明 logo 源重新生成应用、安装器、托盘和 DMG 图标资源，避免 Windows 图标黑边或不透明边缘。
- 改进 Windows 输入源切换：移除高层 DDC 封装，改为直接调用 dxva2 物理显示器接口，并加强 VCP 0x60 重试。
- 保持当前 UI 的按显示器单独切换设计，不改变现有操作逻辑。

### 打包

- 将透明 logo 放回常规的 `src/assets/logo.png` 项目资产路径。
- 已验证 macOS app bundle 指向 `icon.icns`，DMG 构建输入也包含带 alpha 的挂载卷图标。

### 注意事项

- macOS 构建面向 Apple Silicon，要求 macOS 12.0 或更高版本。
- macOS 构建暂未签名/公证；如有需要，请使用右键打开或文档中的 quarantine
  备用命令。
- Windows 构建暂未签名，可能出现 Microsoft SmartScreen 提示。
