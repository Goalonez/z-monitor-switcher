# Z Monitor Switcher v0.1.8

## English

### Highlights

- Improved the Ubuntu tray experience by adding a native tray menu fallback for
  opening controls, showing the main window, and quitting the app.
- Made shortcut recording more reliable by waiting for existing app hotkeys to
  unregister before capturing a new key combination, then restoring hotkeys
  after capture or cancel.
- Cleaned up the external monitor card so the model and serial remain the focus
  without showing a separate vendor row.
- Aligned the brightness and monitor-volume controls in the main window for a
  cleaner Ubuntu layout.

### Documentation

- Simplified the English and Chinese READMEs so they focus on installation,
  supported features, limitations, FAQs, and feedback.

### Notes

- Linux support targets Ubuntu 22.04 X11. Monitor compatibility still depends on
  DDC/CI support, `/dev/i2c-*` permissions, and the desktop environment's tray
  behavior.
- macOS builds still target Apple Silicon and require macOS 12.0 or later.
- macOS builds are not signed or notarized yet; use right-click Open or the
  documented quarantine fallback if needed.
- Windows builds are unsigned and may show Microsoft SmartScreen.

## 简体中文

### 亮点

- 改进 Ubuntu 托盘体验，新增原生托盘菜单入口，可打开快捷控制、显示主窗口和退出应用。
- 提升快捷键录制可靠性：录制前先注销应用已有快捷键，完成录制或取消后再恢复快捷键。
- 精简外接显示器卡片，保留型号和序列号，不再单独显示厂商行。
- 调整主窗口亮度和显示器音量控件的对齐，让 Ubuntu 下的布局更干净。

### 文档

- 简化中英文 README，让内容聚焦安装、支持能力、限制、常见问题和反馈入口。

### 注意事项

- Linux 支持目标是 Ubuntu 22.04 X11。显示器兼容性仍取决于 DDC/CI 支持、
  `/dev/i2c-*` 权限以及桌面环境的托盘行为。
- macOS 构建面向 Apple Silicon，要求 macOS 12.0 或更高版本。
- macOS 构建暂未签名/公证；如有需要，请使用右键打开或文档中的 quarantine
  备用命令。
- Windows 构建暂未签名，可能出现 Microsoft SmartScreen 提示。
