# Z Monitor Switcher v0.1.7

## English

### Highlights

- Added the first Linux / Ubuntu support path, including a Linux DDC/CI backend
  over `/dev/i2c-*`, Linux native controls, Linux post-actions, tray safeguards,
  release packaging, and user documentation.
- Added `Z-Monitor-Switcher.deb` as the Linux installer.

### Linux / Ubuntu

- External monitor DDC/CI control now uses Linux i2c-dev devices through the
  `ddc-i2c` backend.
- Linux system volume uses `wpctl` first and falls back to `pactl`.
- Linux built-in brightness uses writable providers under
  `/sys/class/backlight`.
- Linux keep-awake uses `systemd-inhibit`; post-action sleep and shutdown use
  `systemctl suspend` and `systemctl poweroff` after the existing confirmation
  flow.
- Linux keeps the tray enabled so the app remains reachable after closing the
  main window.

### Documentation

- Updated English and Chinese READMEs with `.deb` installation, I2C permission
  setup, Linux capabilities, limitations, and release artifact names.

### Notes

- Linux support targets Ubuntu 26.04. Monitor compatibility still depends on
  DDC/CI support, `/dev/i2c-*` permissions, and the desktop environment's tray
  behavior.
- macOS builds still target Apple Silicon and require macOS 12.0 or later.
- macOS builds are not signed or notarized yet; use right-click Open or the
  documented quarantine fallback if needed.
- Windows builds are unsigned and may show Microsoft SmartScreen.

## 简体中文

### 亮点

- 新增首版 Linux / Ubuntu 支持路径，包括基于 `/dev/i2c-*` 的 Linux
  DDC/CI 后端、Linux 本机控制、Linux 后置动作、托盘保护、发布打包和用户文档。
- 新增 `Z-Monitor-Switcher.deb` 作为 Linux 安装包。

### Linux / Ubuntu

- 外接显示器 DDC/CI 控制现在通过 `ddc-i2c` 后端访问 Linux i2c-dev 设备。
- Linux 系统音量优先使用 `wpctl`，不可用时回退到 `pactl`。
- Linux 内置屏亮度使用 `/sys/class/backlight` 下可写的 provider。
- Linux 保持唤醒使用 `systemd-inhibit`；后置动作休眠和关机在现有确认流程之后
  调用 `systemctl suspend` 和 `systemctl poweroff`。
- Linux 上保持托盘启用，避免关闭主窗口后无法找回应用。

### 文档

- 更新中英文 README，补充 `.deb` 安装、I2C 权限配置、Linux 能力、限制和发布
  资产名称。

### 注意事项

- Linux 支持目标是 Ubuntu 26.04。显示器兼容性仍取决于 DDC/CI 支持、
  `/dev/i2c-*` 权限以及桌面环境的托盘行为。
- macOS 构建面向 Apple Silicon，要求 macOS 12.0 或更高版本。
- macOS 构建暂未签名/公证；如有需要，请使用右键打开或文档中的 quarantine
  备用命令。
- Windows 构建暂未签名，可能出现 Microsoft SmartScreen 提示。
