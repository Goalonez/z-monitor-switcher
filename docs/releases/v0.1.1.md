# Z Monitor Switcher v0.1.1

## English

### New Features

- **In-app auto-update**: Check for updates, download, and install new versions directly from the settings panel. Updates are fetched from GitHub Releases with cryptographic signature verification.
- **Silent startup check**: The app automatically checks for updates 3 seconds after launch; new versions are indicated with a red badge on the settings icon.

### Notes

- This is the first release with auto-update support. Users on v0.1.0 must manually install v0.1.1 to establish the update chain.
- macOS builds target Apple Silicon and require macOS 12.0 or later.
- macOS builds are not signed or notarized yet; use right-click Open or the documented quarantine fallback if needed.
- Windows builds are unsigned and may show Microsoft SmartScreen.

## 简体中文

### 新功能

- **应用内自动更新**：在设置面板检查更新、下载并安装新版本。更新产物从 GitHub Releases 获取，通过密码学签名验证安全性。
- **启动时静默检查**：应用启动 3 秒后自动后台检查更新；发现新版本时设置图标显示红点提示。

### 注意事项

- 本版本首次支持自动更新。v0.1.0 用户需手动安装 v0.1.1 以建立更新链，后续版本均可应用内更新。
- macOS 构建面向 Apple Silicon，要求 macOS 12.0 或更高版本。
- macOS 构建暂未签名/公证；如有需要，请使用右键打开或文档中的 quarantine 备用命令。
- Windows 构建暂未签名，可能出现 Microsoft SmartScreen 提示。
